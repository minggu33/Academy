position 요소의 위치 지정법

  • position:static; 기본값. 위치 변동 불가.
  • position:relative; 현재 위치에서 ~이동
  • position:absolute; 상위요소를 기준으로 ~위치
    기준이 될 상위요소는 position:relative;를 지정. (기본값이 아니라야 함.)
  • position:fixed; 브라우저를 기준으로 ~에 고정.
    (left/right와 top/bottom값을 지정해야 나타남)
  • position:sticky; relative와 fixed의 믹스.

position:sticky; relative와 fixed의 믹스.

position:sticky;를 적용할 요소는 상위에 body태그 이외에는 다른 요소가 없어야 함

   z-index:n; n값이 클수록 상대적으로 위에 올라옴.
   기본적으로 태그 순서대로 값은 1씩 증가함.
