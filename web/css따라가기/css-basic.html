css-basic

html로 만든 웹문서를 꾸미는 역할.
동시에 여러 문서를 일관된 스타일로 꾸밀 수 있고
동시에 수정도 가능하므로 유지보수에도 유리함.

선택자 { CSS속성 : CSS속성값; }
(주인공) (꾸밈종류)=(꾸밈값).

css 적용법

  • External CSS - 외부 파일 방식.
    CSS소스가 독립되어 존재(abc.css)하며 head안의 link태그로 웹문서와 연결시켜 적용.
    여러 웹문서를 동시에 스타일 적용하고 동시에 수정이 가능.
    <head>
    <link href="abc.css" rel="stylesheet">
    </head>
  • Internal CSS - html문서의 head태그 안에 넣는 방식.
    태그와 CSS소스가 분리되어 적용과 수정은 용이하나 해당 웹문서에만 적용 가능.
    <head>
    <style type="text/css">
    선택자 { CSS속성:CSS속성값; }
    </style>
    </head>
  • Inline CSS - 해당 태그 뒤에 직접 적는 방식
    <tag style="CSS속성:CSS속성값;">
    소스가 길어지고 중복 됨, 일관된 스타일 적용불가, 유지보수 불리.
    3가지 방식 중 css적용 우선 순위가 가장 높은 장점.

css 선택자 종류

  • tag 선택자 (tag) - 강제 적용, 자동 적용(태그가 같으면 같은 스타일)
  • id 선택자 (#abc) - id값이 유일한 값이므로 한번만 적용가능. 구조적 목적에 주로 쓰임
  • class 선택자 (.abc) - class값은 중복적용이 가능. 디자인적 목적에 두루 쓰임.
    class값은 2개이상 조합하여 적용도 가능.
  • 전체 선택자 (*) - 모든 요소를 불러와서 적용
  • ...
