rwd - 반응형웹디자인

 모바일퍼스트 - 작은화면에 맞게 먼저 디자인.
 큰 고정너비 사용 주의, 절대위치 지정시 주의
 (가로스크롤바 생기지 않게)

viewport설정하기

head안에
<meta name="viewport" content="width=device-width, initial-scale=1.0">
를 추가하여 스마트폰에서 내용이 축소되는 것을 방지

CSS3 미디어 유형

  • all - 모든 종류의 장치
  • print - 프린터
  • screen - screen형태의 모든 computer (tablets, smart-phones etc..)
  • speech - 시각장애인용 음성낭독기

미디어쿼리 구문

  @media 장치유형 and (조건:값) {
   선택자 {속성:속성값;
       속성:속성값;
       ...
       }
  }

미디어쿼리의 조건

  • 뷰포트의 너비와 높이 (width and height of the viewport)
  • 장치의 너비와 높이 (width and height of the device)
  • 방향 - 태블릿, 휴대폰이 가로 또는 세로 모드 (orientation landscape or portrait)
