padding - 내부여백

테두리를 기준으로 안쪽에 양사방 적용되는 여백.

padding의 값 갯수에 따른 적용법

  • padding : n ; 양사방 동일
  • padding:n n; 상하 좌우가 서로 동일
  • padding : n n n ; 상 좌우동일 하
  • padding : n n n n ;

padding의 방향별 적용 속성

  • padding-top:n ;
  • padding-right:n ;
  • padding-bottom:n ;
  • padding-left:n ;
