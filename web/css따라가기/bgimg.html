background 배경

  • background-color:색상값; 배경색상
  • background-image:url("이미지경로"); 배경이미지

배경이미지의 속성들

  1. background-repeat: ; 반복 지정
    • repeat 기본값. 양사방 반복
    • repeat-x 가로로만 반복
    • repeat-y 세로로만 반복
    • no-repeat 반복 안 함
  2. background-size: ; 크기 지정
    • cover 이미지 하나로 영역을 다 덮기(이미지가 잘려나갈 수도 있음)
    • contain 이미지 하나를 온전히 보이면서 영역을 덮기
    • Xn Yn 가로 세로 순으로 직접 값을 적용
  3. background-position: ; 위치 지정
    • left/center/right top/center/bottom 방향값을 조합하여 적용
    • Xn Yn 가로 세로 순으로 직접 값을 적용
  4. background-attachment: ; 스크롤여부 지정
    • scroll
    • fixed
    • local


















background-attachment: ; 스크롤여부 지정

 scroll은 브라우저 스크롤바에는 스크롤, 
         자신 영역의 스크롤바에는 고정.  
 fixed는 브라우저 스크롤바와 
         자신 영역의 스크롤바에 모두 고정.  
 local은 브라우저 스크롤바와 스크롤, 
         자신 영역의 스크롤바에 모두 스크롤.
