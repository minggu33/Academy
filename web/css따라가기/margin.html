테두리를 기준으로 밖에 양사방 적용되는 여백.

margin의 값 갯수에 따른 적용법

  • margin: n ;
  • margin: n n ;
  • margin: n n n ;
  • margin: n n n n;

margin의 방향별 적용 속성

  • margin-top : n ;
  • margin-right: n ;
  • margin-bottom: n ;
  • margin-left: n ;

inline에 margin을 적용

1. 자동으로 줄이 안바뀌는(한 줄에 나란히 올 수 있는)태그를 inline요소라고 하며
크기 적용 불가하고 상하 margin적용 불가!
b tag-1b tag-2b tag-3

2. 자동으로 줄이 바뀌는(한 줄에 나란히 올 수 없는)태그를 block요소라고 하며
크기 적용 가능하고 모든 방향의 margin적용 가능.
