form으로 사용자가 서버에 정보를 쉽게 전달할 수 있는 페이지를 구성한다.
action속성으로 서버의 파일을 지정하고 method속성으로 전달 방식을 지정.
(일반적으로는 post, 보안상의 문제없고 데이터의 길이가 짧을 때는 get)

input type="text" 한줄 글상자

input상자에 입력할 정보에 대한 설명문은 label태그로 지정하고 for의 속성값을 input태그의 id속성값과 서로 일치시켜 글상자와 설명문을 연결시킴.
size속성으로 상자의 가로 길이를 지정하고
maxlength속성으로 입력데이터의 최대길이를 제한할 수 있다. (최소는 minlength)

input type="hidden" 숨긴 데이터(사용자에게 보이지 않는 데이터 전송용)

input type="file" 파일 전송용

input type="submit/reset/button" 전송/초기화/일반 버튼

input type="submit/reset"은 폼의 정보를 전송하거나 초기화하는 역할의 버튼,
input type="button" 폼과는 무관한 일반 버튼이 생성됨.
버튼의 글자는 value속성으로 지정.

input type="image" 전송하는 역할의 그림

src속성으로 그림파일의 경로를 지정

button ... /button 전송/초기화/일반 버튼

button은 기본적으로 폼의 정보를 전송하는 역할의 버튼임.
type="reset"으로 폼의 정보를 초기화할 수 있고,
type="button"으로 폼과는 무관한 일반 버튼이 될 수 있다.
버튼의 글자는 시작과 종료 태그 사이에 입력.
