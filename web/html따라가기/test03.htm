list - 목록

목록의 종류

  • <ul> - 순서 없는 목록 (unordered list)
  • <ol> - 순서 있는 목록 (ordered list)
  • <dl> - 정의 목록 (description list)

1. 순서 없는 목록 - ul

web의 기본 기술 3가지

  • html - 웹문서의 구조를 담당
  • css - 웹문서의 디자인을 담당
  • javascript - 웹문서의 액션을 담당
  • red
  • green
  • blue

  • 돌체라떼
  • 녹차라떼
  • 딸기쥬스
  • 팥절미 밀크쉐이크
