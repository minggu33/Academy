table로 표 만들기

table 표를 구성하는 태그
tr 줄을 구성하는 태그
th 제목 칸을 구성하는 태그 (볼드체, 중앙정렬)
td 일반 칸을 구성하는 태그

가로형 표

name lee hee ju lee eun ji
birthday 1990.09.09 1990.10.15

세로형 표

name birthday
lee hee ju 1990.09.09
lee eun ji 1990.10.15

칸 합치기

  1. td colspan="n" 가로 방향으로 n개의 칸을 하나로 합치기
  2. rowspan="n" 세로 방향으로 n개의 칸을 하나로 합치기
