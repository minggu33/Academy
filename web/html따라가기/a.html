a태그로 link걸기

<a href="url(이동할 곳의 주소)"> ... </a>

a태그에 href속성으로 주소(경로)를 지정한다.
링크가 걸린 텍스트는 밑줄이 나타나고
마우스 오버시 커서의 모양이 바뀌고
색상은 3가지 형태중 하나로 나타남.
▶초기링크색 - 파랑
▶방문한링크색 - 보라
▶활성화시링크색 - 빨강

link의 경로 지정

  1. 상대경로 - html문서의 위치에 따라 달라지는 경로
  2. 절대경로 - 항상 같은 경로

1.상대경로로 link걸기

form-test 이미지 수업

2.절대경로로 link걸기

w3schools.com daum

target속성으로 링크 여는 방식 지정하기

daum(새 탭에서 열기)

target 속성값

  • target="_self" (기본값)현재 위치에 열기
  • target="_blank" 새 탭에서 열기
  • target="_parent" 한단계 상위창에 열기
  • target="_top" (프레임으로 나뉨없이)전체 창에 열기
네이놈을 새 탭에서 열기

링크로 보여줄 수 있는 종류 - 브라우저가 처리 가능한 파일(종류)인 경우 바로 보여줌

  1. 외부사이트 (url)
    pixabay(이미지무료1) pexels(이미지무료2)
  2. 웹페이지 (*.htm or *.html) 이미지수업보기
  3. 이미지파일 (*.jpg / *.gif / *.png)
    drop.jpg arrow-up.png arrow.gif
  4. 동영상파일 (*.ogg / *.mp4 / *.webm)
    trees.mp4
  5. 텍스트파일 (*.txt)
    이클립스다운받기.txt

링크로 보여줄 수 없는 종류 - 브라우저가 처리 불가한 파일(종류)인 경우 다운로드 됨

img.zip zip, hwp 등은 다운로드 됨

이미지에 링크 걸기 (링크 걸린 이미지)

 <a href="보여줄 내용의 경로" title="툴팁 내용"> 
  <img src="(클릭할) 이미지의 경로"> 
 </a>
 
                 *title속성, 툴팁은 다른 태그에도 사용 가능.

- 이미지(로고) 클릭시 해당 사이트 보여주기


- 이미지(썸네일) 클릭시 이미지 크게 보여주기


- 이미지(영상의 캡쳐본) 클릭시 동영상 보여주기
