html5에서 추가된 폼요소들

input type="time" 시간(시분)을 지정

input type="date" 날짜를 지정

input type="datetime-local" 날짜, 시간(년월일시분)을 지정

input type="week" 날짜(연도,주)을 지정

input type="month" 날짜(연도,월)을 지정

input type="tel" 연락처(전화번호)을 지정

모바일브라우저에서 가상키패드가 숫자형식으로 바뀔 수 있고 전화걸기 가능

input type="email" 연락처(이메일주소)을 지정

모바일브라우저에서 가상키패드가 영문(.com) 포함하는 형식으로 바뀔 수 있고 브라우저에서 알맞은 이메일형식(@여부,뒷자리)인지를 체크함

input type="url" 연락처(웹사이트주소)을 지정

모바일브라우저에서 가상키패드가 영문(https:// 등) 포함하는 형식으로 바뀔 수 있고 브라우저에서 알맞은 형식(https:// 등 프로토콜 여부)인지를 체크함

input type="number" 숫자 형식을 지정

숫자값을 포함하는 형식. 증감버튼이 생성. 최소값과 최대값을 속성으로 지정할 수 있고 증감 간격을 속성으로 지정할 수 있다.


min="0" 지정
max="10" 지정
step="12" 지정

input type="range" 범위내 값을 지정

input type="color" 컬러값을 지정

html5에서 추가된 속성들

autofocus

autocomplete="on/off" - 자동완성 기능을 켜거나 끔


placeholder - 입력예시를 보여주거나 설명문을 넣어줌

readonly - 읽기만 가능한 요소로 지정

disabled - 사용불가한 요소로 지정

required - 필수입력 요소로 지정
