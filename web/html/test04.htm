table로 표 만들기

table 표를 구성하는 태그
tr 줄을 구성하는 태그
th 제목칸을 구성하는 태그(볼드체, 중앙정렬)
td 일반칸을 구성하는 태그

가로형 표

name lee hee ju lee eun ji
birth day 1990.09.09 1989.09.30

세로형 표

name birthday
lee hee ju 1990.09.09
lee eun ji 1989.09.30
  1. td(th) colspan="n" 가로방향으로 n개의 칸을 하나로 합치기
  2. td(th) rowspan="n" 세로방향으로 n개의 칸을 하나로 합치기
