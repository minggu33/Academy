0310 DB


<제약조건>

1.Not Null - 컬럼에 null값이 들어오는 것을 막아줌
2.Unique - 중복된 값이 들어오는 것을 막아줌
3.Primary key(기본키) - not null + unique
		     - 테이블당 한번 선언 가능함!
4.Foreign key(외래키) - 특정 테이블의 특정 컬럼을 참조함
		     - 참조하는 컬럼의 값만 사용 가능함
5.Check - 해당 컬럼이 만족해야하는 조건문을 지정함


<명령어>

select table_name  
from user_tables;  -> 내 소유의 테이블명 확인하는 명령어

col department_name format a20 -> 원하는 컬럼 사이즈 편집 a뒤에 사이즈 크기(숫자나 날짜 들어있는
				  컬럼은 조절하면 안됨.문자가 들어있는 컬럼만 사이즈 조절o)

l - 버퍼에 있는 마지막 쿼리구문 확인
/ - 재실행

commit; -> insert나 update 등 새로운 작업을 한 후에 마지막에 저장을 하는 명령어 

rollback; -> 작업한 내용 저장하지 않고 취소하는 명령어
