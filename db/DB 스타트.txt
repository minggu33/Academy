바탕화면 오른쪽마우스 클릭 터미널 열기

<DB 시작 및 사용자 접속>
1. ORCL 가상 PC 시작
2. Linux 로그인 -username : oracle / password : oracle
3. 터미널 창 열기 -바탕화면 오른쪽마우스 클릭 터미널 열기
4. 리스너 활성화  		$] lsnrctl start			$] <- 리눅스단이라는 의미
5. sqlplus 프로그램 실행 	$] sqlplus /nolog
6. sys 사용자로 DB 로그인	SQL>conn sys as sysdba
7. DB 시작		SQL>startup
8. hr 사용자로 DB 로그인 	SQL>conn hr/hr


리눅스 명령어

pwd = 지금 현재 나의 위치 확인하는 명령어

lsnrctl start = 리스너 스타트, 활성화하는 명령어

sqlplus /nolog = sql 활성화 하는 명령어

conn sys/oracle as sysdba = db에 sys사용자로 접속     *db켤때 sys사용자로 접속해서 켜야함
connect 시스아이디/비밀번호 as sysdba

show user = 지금 접속된 사용자 확인

startup = db 켜는 명령어

conn 유저명/패스워드 = 다른 유저로 접속할때

!clear = 화면 위로 올리기

alter user hr            해서 엔터치고
identified by hr;    = 새로운 패스워드 설정(180일마다 만기되서 주기적으로 바꿔줘야함) 새로운 비번도 hr

ed = 버퍼에 들어있는 제일 마지막에 작성한 쿼리구문을 수정할때 쓰는 명령어

그이후에 들어가서 i 누르면 편집모드 가능, 버퍼에는 세미콜론 작성하면 안됨.

다 작성후에 esc키 한 다음에 1)저장하고 나가는 키 :wq 2) 수정하기 전 원래대로 돌아가는 키 :q! 

마지막에 바뀐 구문 실행은 / 적고 엔터   <- 여기까지 해야 수정되서 저장되고 실행 끝

desc 테이블명 = 테이블구조 확인할 때 사용하는 명령어
